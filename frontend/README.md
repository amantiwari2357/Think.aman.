## Features
- Industry-specific user profiles
- Post creation (memes, jokes, information)
- Problem-solving requests
- Real-time notifications
- Follow/following system
- Social interactions (likes, comments)
- User dashboards

## Technologies Used
- React with TypeScript
- React Router for navigation
- TanStack Query for data fetching
- React Hook Form for form management
- Shadcn UI components
- Tailwind CSS for styling
- Lucide React for icons
- Date-fns for date formatting
- Sonner for toast notifications

## Setup Process
1. Clone the repository
2. Install dependencies with `npm install`
3. Start the development server with `npm run dev`
4. Access the application at http://localhost:5173

## Working Process
The platform allows users to:
1. Register with their industry
2. Create and view posts specific to their industry
3. Post problems and connect with experts
4. Follow other users and receive real-time notifications
5. Like, comment, and share content
